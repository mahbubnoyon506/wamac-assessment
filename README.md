# PPA - Master Focus & Deep Work

This project is a high-performance web application designed to help users eliminate procrastination and train their brains for deep work. Built with a modern tech stack, it features a structured course curriculum, interactive testimonials, and a results-driven results-driven blueprint.

---

## Local Setup

Follow these steps to get the project running on your local machine:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/mahbubnoyon506/wamac-assessment.git
    ```
2.  **Navigate to the directory:**
    ```bash
    cd wamac-assessment
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Start the development server:**
    ```bash
    npm run dev
    ```
5.  **Open the app:**
    Visit `http://localhost:3000` in your browser.

---

## Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS with custom responsive typography
- **Components:** shadcn/ui & Lucide React
- **Animations/Sliders:** react-slick
